Що це і для чого?

Якщо потрібно швидко створити простий HTML документ, а на форматування та написання CSS немає часу - можна використати уже готові стилі форматування елементів

Як підключити?

Додати файл penguins-ui.css до свого проекту та підключити його в HTML файлі з допомогою тегу <link>:

<link rel="stylesheet" href="penguins-ui.css">

Скачати penguins-ui.css

Як використовувати?

Потрібно додати CSS клас p-ui до тегу <body>. Після цього CSS стилі бібліотеки будуть автоматично застосовані до елементів сторінки. Весь контент документу повинен знаходитись всередині блоку з класом main-container

Базова структура документу

Для вирівнювання контенту по центру документу потрібно створити блок-контейнер і додати йому CSS клас main-container

Схема базової структури документу:

За замовчуванням документ матиме одноколоночну структуру. Елемент з класом main-container має властивості display: flex та flex-direction: column

Приклад базової HTML розмітки

<body class="p-ui">
<main class="main-container">
<section>
<h2>Lorem ipsum</h2>
<hr>
<p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Ea minima sequi sunt voluptatibus, rem praesentium dolorum. </p>
</section>
</main>
</body>

Розмітка документу на основі рядків і колонок (flexbox)

Для забезпечення необхідної структури розташування блоків з контентом, а також забезпечення адаптивності можна використовувати рядки та колонки, до 12 колонок у кожному рядку.

Створення структури блоку з колонками

Щоб додати контейнер для колонок необхідно створити блок з классом f-row

Для створення колонки необхідно додати блочний елемент з відповідним класом. Бібліотека використовує принцип mobile first, тому імена класів мають наступну структуру:

f-col - ширина блоку буде займати весь доступний простір блоку з класом f-row, якщо блоків з класом f-col буде декілька, вони поділять доступний простір між собою на рівні частини.

f-col-4 - для зміни співвідношення ширини блоків до імені класу додається цифра, яка показує скільки з 12 доступних колонок буде займати блок.

Зміна ширини колонок в залежності від розміру екрану

Для зміни ширини блоків в залежності від розміру екрану в імені класу використовуються префікси -tab- (tablet) для планшетів (ширина екрану більше 768px), та -dt- (desktop) для компьютерів (ширина екрану більше 1024px), таким чином імена класів будуть мати наступний вигляд:
f-tab-col-8 - для планшетів,
f-dt-col-4 - для настольних ПК.
Блок який має дані класи буде займати 8 колонок з 12 на планшеті та 4 колонки з 12 на компьютері.

Примітки до форматування коду:

  • клас"code-block" додає властивість display: block
  • для виділення рядків використовується клас "code-string"
  • для виділення CSS селекторів використовується клас "code-class"
  • для створення відступів використовується клас "code-margin"
  • для виділення вмісту HTML тегу використовується клас "code-tag-inner"

Приклад HTML розмітки коду:

<pre>
    <span class="code-class">.p_ui blockquote</span> {
        font-size:  <span class="code-string">18px</span>;
        margin: <span class="code-string">32px 16px 32px 48px</span>;
        padding: <span class="code-string">32px 8px 32px 16px</span>;
        font-family: <span class="code-string">'Inter', sans-serif</span>;
        font-weight: <span class="code-string">700</span>;
    }
</pre>

Результат:

CSS позиціонування. Різниця між relative та absolute

Я звичайний блок з position: static як і у всіх блоків за замовчуванням.
Я його елемент
вищого рівня --->
Я блок з position: absolute; left: 200px; bottom: 10px;. Я не впливаю на оточуючі блоки, та зміщуюся відносно елементу вищого рівня на відстані, вказані за допомогою властивостей left, right, top, bottom. Елемент вищого рівня має отримати властивість position: relative, інакше я буду позиціонуватися відносно всієї сторінки взагалі.
Я блок з position: relative; right: 50px; top: 20px;. Я впливаю на оточуючі блоки, та зміщуюся відносно своєї позиції на відстані, вказані за допомогою властивостей left, right, top, bottom.
